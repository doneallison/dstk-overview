The Data Science Toolkit

A Basic Overview and Case Study

By Don Allison

Background

  • http://www.datasciencetoolkit.org/
  • Specialized Linux distribution
    • Built-in open datasets
    • Simple interface
  • Created by Pete Warden

Features

Geocoding

  • Coordinates of physical address
  • Coordinates of IP address
  • Political boundaries for coordinates

Geostatistics

  • Statistics for a coordinate pair
    • Geographic: Elevation, temperature, precipitation, etc.
    • Demographic: Population density, housing units, racial makeup, etc.

Document parsing

  • Image/PDF to plaintext
  • HTML to plaintext

Text processing

  • Name and attributes of people mentioned in text
  • Coordinates of places mentioned in text
  • UNIX time for times mentioned in text
  • Sentiment analysis

Benefits

Free and open source

  • Source code is available on GitHub
  • Built with free and open-source software
  • Uses free and open datasets

Online or offline

  • Query the DSTK server directly...
    • Flexibility
    • Easy prototyping
  • ...or set up your own server
    • Low latency
    • Better data security

Customizable

You have several options for setting up your own server

  • Ready-to-go EC2 instance
  • Vagrant box
  • Open Virtualization Format
  • Roll your own

Accessible

Libraries are available for

  • Command line
  • Excel
  • JavaScript
  • R
  • Ruby
  • Python

Simple Example

Command Line Tools (Linux/OS X)

DSTK comes with Python scripts that provide command line access to the server's APIs. You can configure whether these tools should query a remote or local server.

$ wget "http://www.datasciencetoolkit.org/python_tools.zip"
$ unzip python_tools.zip
$ cd python
$ sudo ./install
$ cd ~
$ street2coordinates "1095 6th Ave New York, NY"
"1095 6th Ave New York, NY",0.87,1095,New York,NY,-73.984118,USA,United States,36061,US,40.754842,6 Ave, 1095 6 Ave
$ ip2coordinates "67.169.73.113"
67.169.73.113,San Francisco,CA,415,-122.438102722,USA,United States,94114,807,US,37.7587013245

Case Study: Trouble in Paradise

Bulk Geocoding Incident Data

  • Task: Geocode thousands of physical addresses in a crime incident dataset for Honolulu
  • Challenges
    • Transaction limits
    • Latency
    • Data security
      • Not really in this case, though, since it's open data
  • Solution: DSTK and Visual Basic for Applications (VBA)

Step One: Obtain DSTK

  • Download the .ova version (~19.8GB) of DSTK
  • Install VirtualBox
  • Import the .ova file into VirtualBox
    • File > Import Appliance
  • Boot DSTK
    • Default username and password are vagrant

Step Two: Configure DSTK

  • Determine the IP of the "guest" machine
    • In DSTK, type ifconfig and find inet address
    • Probably 10.0.2.15
  • Forward that IP to localhost on the "host" machine
    • In VirtualBox, select Machine > Settings > Network > Advanced > Port Forwarding
    • Forward 10.0.2.15:80 (guest) to 127.0.0.1:8000 (host)
  • Verify that visiting 127.0.0.1:8000 on the host machine resolves to DSTK

Step Three: Create VBA Scripts

' Dependencies: Microsoft WinHTTP Services, version 5.1

Public Function getGeocode(address As String) As String

Dim MyRequest As New WinHttpRequest
Dim URL As String

URL = "http://localhost:8000/street2coordinates/" & address
MyRequest.Open "GET", URL
MyRequest.send
getGeocode = MyRequest.responseText
    
End Function

Step Three: Create VBA Scripts

Public Function getLatitude(geocode As String) As String
    If InStr(geocode, "latitude") = 0 Then
        getLatitude = "None"
    Else
        getLatitude = Mid(geocode, InStr(geocode, "latitude") + 10, InStr(Mid(geocode, InStr(geocode, "latitude") + 10), ",") - 1)
    End If
End Function
Public Function getLongitude(geocode As String) As String
    If InStr(geocode, "longitude") = 0 Then
        getLongitude = "None"
    Else
        getLongitude = Mid(geocode, InStr(geocode, "longitude") + 11, InStr(Mid(geocode, InStr(geocode, "longitude") + 11), ",") - 1)
    End If
End Function
